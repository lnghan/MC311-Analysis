## DATA205-Capstone-Project

This project explores the MC311 dataset from data.montgomery. It specifically focuses on three service areas: Highway Services, Tree Maintenance, and Code Enforcement, as well as five randomly chosen ZIP code in Montgomery County, Maryland: 20833 (Brookeville), 20866 (Burtonsville), 20904 (Silver Spring), 20906 (Silver Spring), 20850 (Rockville)

I aim to identify patterns in demand, response time, and potential disparities. Ultimately, this project could help highlight opportunities for improving efficiency and equity in local government services.
